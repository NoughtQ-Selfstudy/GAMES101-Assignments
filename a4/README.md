# 说明

所有评分点均完成。

- 提交格式正确
- De Casteljau 算法
    - `recursive_bezier` 函数
        - 若只有一个控制点，则立即返回该控制点
        - 否则准备一个大小比原来小 1 的控制点序列，每两个相邻的旧控制点生成一个新的控制点
        - 注意前一个控制点权重为 `1-t`，后一个控制点权重为 `t`
        - 返回将新控制点序列作为参数的运算结果
    - `bezier` 函数
        - 设置步长（我设置步长为 1e-4）
        - 对于每一步，调用 `recursive_bezier` 计算点的坐标，并仿照 `naive_bezier` 在相应像素点上色
- 奖励部分：实现对 Bézier 曲线的反走样
    - 类似双线性插值，找出离采样点最接近的 4 个像素点（注意越界处理）
    - 根据采样点离像素点的距离，为像素点分配颜色，距离 > 1 的不分配颜色
    - 像素点颜色为离它最近的采样点为它赋予的颜色
